## 🔧 기술 스택
- API : `Kakao API`, `DAUM 주소 검색 API`, `Kakao MAP API`
- Language : `JAVA(11)`, `JavaScript(1.5)`
- Library & Framework : `Spring(5.2.7)`, `JUnit(4.12)`, `Spring Security(5.2.7)`
- Database : `Oracle(11g)`
- Target : `Web Browser`
- Tool : `Spring Tool Suite 3.9.18.RELEASE`
- ETC : `Git`
